# Quiz Portal

An online quiz portal developed using Django/Flask for conducting quizzes and tests digitally.  
This project allows users to attempt quizzes online and get instant results, while the admin can manage questions, users, and scores.

---

# Features

## Admin Features
- Admin Login
- Add/Edit/Delete Questions
- Manage Quiz Categories
- View User Scores
- Manage Users

## User Features
- User Registration and Login
- Attempt Quizzes
- Instant Result Generation
- Timer-Based Quiz
- View Previous Scores

# Project Modules

## Authentication Module
Handles user registration and login.

## Quiz Management Module
Allows admin to create and manage quizzes.

## Quiz Attempt Module
Users can attempt quizzes and submit answers.

## Result Module
Calculates and displays quiz scores instantly.

---

# How It Works

1. User registers and logs into the portal.
2. Admin adds quiz questions.
3. User selects a quiz category.
4. Questions are displayed.
5. User submits the quiz.
6. System calculates score automatically.
7. Final result is displayed.

# Installation


# For Django

```bash

```

```bash
pip install flask
python app.py
```
